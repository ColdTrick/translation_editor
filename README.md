Manage translations

Note
----

If you wish to upgrade this plugin from a version < 0.5.3 please first install the latest release before 2.0.

Features
-----------
- Edit translations
- Search translations
- Assign multiple translators
- Add custom languages
- Add custom keys
- Caching of language files
